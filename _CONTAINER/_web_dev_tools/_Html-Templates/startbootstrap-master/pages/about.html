History

Start Bootstrap was founded by, and is maintained by, David Miller - a web developer based in the Central Florida area. The website was launched on August 19th, 2013. Since then, over 4,000,000 users have accessed our themes, templates, snippets, and guides! All of the free content on Start Bootstrap is MIT Licensed and free to download - that means you can use our themes and templates for any purpose, even for commercial projects.

Feedback

If you want to get into contact with Start Bootstrap, the easiest way to do so is by following us on Twitter @SBootstrap.

Custom Projects

If you have a specific UI/UX project in mind, you can hire the Start Bootstrap team to help create a custom design for your project.
