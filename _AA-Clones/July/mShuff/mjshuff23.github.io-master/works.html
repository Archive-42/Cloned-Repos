BUILD-A-BODY

Build-A-Body is a project I've created myself in the last month. It's a health and fitness app focused on the value of hard work and a healthy diet. It's written in React JSX with utilizing a global Redux store and an Express backend. I also incorporated polymorphic associations through Sequelize in a PostgreSQL database


KLEVERNOTE

Klevernote is a note taking app written with React.js and Redux.js as on the frontend and Python and SQLAlchemy on the backend. I've been collaborating with others on this project, and it utilizes Material-UI and React-Quill to provide a rich, seamless experience for the UX.


DISCHORD

Dischord is a real-time chat app for gamers and classrooms alike utilizing React.js, Redux.js, Socket.io and Express.js. This is a solo project I've been working on. It is designed to have a smooth UX and utilizes a PostgreSQL database which I incorporate many-to-many join table relationships. I've also incorporated Material-UI and security measures by implemented CORS and JSON Web Tokens


CORE DUMP

Core Dump is a site for developers to get all their programming-related questions answered by colleagues and peers, utilizing native JavaScript, Sequelize, and JSON Web Tokens. Incorporated PUG templates and AJAX calls, along with filter search functionality.
