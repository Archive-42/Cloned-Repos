Submit a Resource

To submit a resource for review, please send an email to feedback@startbootstrap.com with the following information:

  • The name/title of the resource
  • A URL for the resource
  • A brief description of the resource

Each submission is reviewed for quality and relevance. If you have a quality resource that is relevant to Bootstrap then it will most likely be added to the resources page!
