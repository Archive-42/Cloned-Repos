Privacy Information

Please note that this is an example app created for demonstration and testing purposes. If you login to this app, your 'display name' from your social profile will be stored temporarily in the app's database. This database is cleared of all new user registrations on a regular basis. If you would like your name removed from the database immediately after testing the app, please email admin@londonacademyofit.co.uk.
