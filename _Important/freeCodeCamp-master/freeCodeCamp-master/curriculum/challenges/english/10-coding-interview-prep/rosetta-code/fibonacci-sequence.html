–description–

Write a function to generate the nth Fibonacci number.

The nth Fibonacci number is given by:

Fn = Fn-1 + Fn-2

The first two terms of the series are 0 and 1.

Hence, the series is: 0, 1, 1, 2, 3, 5, 8, 13…

–solutions–

function fibonacci(n) {
  let a = 0, b = 1, t;
  while (--n >= 0) {
    t = a;
    a = b;
    b += t;
  }
  return a;
}
