General customization for d3.js




This section aims to describe how to improve the look of your d3.js chart. It does not target any particular chart type, but give general examples about axis, legends, colors, annotation, small multiple and more.

Axis

D3.js offers built-in function to add axis to your chart. The first step is to build a scale that allows to transform a numeric value to a coordinate on the plot. The second is to call axisBottom() or axisLeft() to build the axis. Browse the example below to understand the different types of existing scale, how to build axis and how to customize them.
