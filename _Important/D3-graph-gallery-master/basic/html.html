What is html




This is a very very short intro to what html is. I highly recommend to spend a few hours reading the w3 school section on html before trying to learn d3.js


Back to the gallery

This is a very basic piece of html. Basically it is composed by different elements. Like p tells that you're gonna write a paragraph.

← Edit me! 
<!DOCTYPE html>
<meta charset="utf-8">

<!-- Add a line of text -->
<p id="mytext">Hello world!</p>
<script src="https://d3js.org/d3.v4.js"></script>

And a bit of Javascript on top of it:

← Edit me! 

<script>
d3.select("#mytext").style("color", "blue")
</script>
