Destructuring

A js expression that allows to unpack values from arrays or properties from objects into distinct variables. Full reference 

// destructuring an array:
const [a, b] = [10, 20]
console.log(b) // 20

// destructuring an object:
const obj = { c: 3, d: 4, e: 89 }
const { c, d } = obj
console.log(d) // 4

Note that you can also rename directly

const otherObj = {f: 78, g: 12};
const {f: newNameForF, g: newNameForG} = otherObj;
console.log(newNameForF); // 78

CSS selector combination

A CSS selector allows to select a specific part of the DOM to give it some style. For instance, make the background red for the class called "classA"

.classA {
  background-color: red;
}

You can do the same with ids (#classA)or with html elements ( p or h1 ).

It is also possible to combine selectors and I always forget how it works

  • Grouping: .class1, .class2, .class3 { ... } select those 3 classes.
  • Descendant: .class1 .class2 {...} select class2 when it is inside a class1.
  • Adjacent sibling: .class1 + .class2 select class2 when it comes right after class1 at the same level.
  • General sibling: .class1 ~ .class2 select class2 when at the same level than class1
  • Child: .class1 > .class2 select class2 when it comes right after class1 as a children.

See more in this Cheatsheet.

Spread syntax ...

The spread syntax allows to merge or clone object and concatenate arrays. Full reference

→ Merge: Note that if a property exists in several objects, its final value will be the last observed:

const obj1 = { a: 1, b: 2 }
const obj2 = { a: 1, b: 4 }
console.log({ ...obj1, ...obj2 }) // { a: 1, b: 4 }

→ Clone: shallow cloning here.

let obj = { a: 2 };
let clone = { ...obj };

→ Concatenate: For arrays

let arr1 = [0, 1, 2];
let arr2 = [3, 4, 5];
let concatArray = [...arr1, ...arr2];
console.log(concatArray) // [0, 1, 2, 3, 4, 5]
